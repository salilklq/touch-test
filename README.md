# 右手机械手控制程序 (test.py)

## 环境要求

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：Python 3.8 及以上
- **硬件**：机械手通过串口（USB 转 RS485）连接电脑

## 安装依赖

本项目唯一依赖的第三方包是 `pyserial`（注意：包名是 **pyserial**，不是 serial）。

```bash
pip install pyserial
```

如果你的电脑上有多个 Python 版本，建议使用：

```bash
python -m pip install pyserial
```

### Windows 一键修复环境

如果安装遇到问题（比如装了错误的 `serial` 包），可以双击运行项目自带的修复脚本：

```
fix_serial_env.bat
```

它会自动查找 Python、卸载错误的 `serial` 包、安装正确的 `pyserial`。

## 串口确认

将机械手通过 USB 转 RS485 连接电脑后，先确认串口号：

- **Windows**：打开设备管理器 → 端口（COM 和 LPT），找到对应的 COM 口（如 COM3、COM19）
- **macOS/Linux**：终端执行 `ls /dev/tty.*` 或 `ls /dev/ttyUSB*`

也可以用 Python 列出所有可用串口：

```bash
python -c "import serial.tools.list_ports; [print(p.device, p.description) for p in serial.tools.list_ports.comports()]"
```

## 使用方法

### 命令格式

```
python test.py <命令> <串口端口> <速度值> <力矩值> [循环次数]
```

| 参数       | 说明                         | 范围      | 示例   |
| ---------- | ---------------------------- | --------- | ------ |
| 命令       | `test`（点动）或 `loop`（循环） | -         | test   |
| 串口端口   | 机械手连接的串口号            | -         | COM3   |
| 速度值     | 所有手指的运动速度            | 0 - 1000  | 300    |
| 力矩值     | 所有手指的力矩大小            | 0 - 1000  | 300    |
| 循环次数   | 仅 loop 模式可选，不填则无限循环 | 正整数    | 5      |

---

### 模式一：交互式点动模式 (test)

手动输入 6 个手指的位置值，每次回车立即执行。

```bash
python test.py test COM3 300 300
```

也可以省略 `test`，直接写串口和参数（兼容旧格式）：

```bash
python test.py COM3 300 300
```

进入后会看到提示：

```
> 请输入6个手指的位置值（空格分隔），输入q/quit退出：
```

输入 6 个 0-1000 的整数，空格分隔，按回车发送。6 个值的顺序为：

| 序号 | 手指         |
| ---- | ------------ |
| 1    | 大拇指翻转   |
| 2    | 大拇指弯曲   |
| 3    | 食指弯曲     |
| 4    | 中指弯曲     |
| 5    | 无名指弯曲   |
| 6    | 小拇指弯曲   |

**示例操作：**

```
> 500 500 0 0 0 0        # 大拇指翻转500 + 弯曲500，其余不动
> 0 0 1000 1000 1000 1000 # 四指完全弯曲，大拇指不动
> 1000 1000 1000 1000 1000 1000  # 所有手指完全弯曲（握拳）
> 0 0 0 0 0 0             # 所有手指归零（张开）
> q                       # 退出程序，自动归零
```

---

### 模式二：循环测试模式 (loop)

按预设的动作序列自动循环执行，用于反复测试机械手动作。

```bash
# 无限循环
python test.py loop COM3 300 300

# 循环 5 轮后自动停止
python test.py loop COM3 300 300 5
```

预设的动作序列（定义在代码 `run_loop_mode` 函数中）：

| 步骤 | 动作说明         | 位置值                          | 延时    |
| ---- | ---------------- | ------------------------------- | ------- |
| 1    | 大拇指翻转       | [1000, 0, 0, 0, 0, 0]          | 0.5 秒  |
| 2    | 大拇指弯曲       | [1000, 1000, 0, 0, 0, 0]       | 0.5 秒  |
| 3    | 四指弯曲复位     | [0, 0, 0, 0, 0, 0]             | 0.5 秒  |
| 4    | 四指弯曲         | [0, 0, 1000, 1000, 1000, 1000] | 1.0 秒  |
| 5    | 全部归零         | [0, 0, 0, 0, 0, 0]             | 1.0 秒  |

如需修改动作序列，编辑 `run_loop_mode` 函数中的 `action_sequence` 列表即可。

**退出方式：**

- 输入 `q` 并回车
- 按 `Ctrl + C`

退出时程序会自动将所有手指归零。

## 协议说明

- **通信协议**：Modbus-RTU
- **波特率**：115200
- **从机地址**：默认为 2（可在代码中修改）
- **功能码**：
  - `06`：写单个寄存器（设置速度、力矩）
  - `10`（0x10）：写多个寄存器（一条报文同时设置 6 个手指位置）
  - `03`：读寄存器（验证写入结果）
- **数值范围**：所有位置、速度、力矩值均为 0 - 1000

### 寄存器地址映射

| 寄存器地址 | 功能             |
| ---------- | ---------------- |
| 0 - 5      | 位置控制（f1-f6）|
| 6 - 11     | 速度控制         |
| 12 - 17    | 力矩控制         |
| 47 - 57    | 厂家专用（禁止操作）|

## 常见问题

### 1. 提示 `ModuleNotFoundError: No module named 'serial'`

安装了错误的包。执行：

```bash
pip uninstall serial
pip install pyserial
```

### 2. 串口打开失败

- 检查串口线是否连接
- 确认串口号是否正确
- 确认没有其他程序占用该串口（如串口调试助手）

### 3. 写入后验证失败（WARN）

- 检查波特率是否为 115200
- 检查从机地址是否匹配（右手默认为 2）
- 检查机械手是否上电
